上屏汉字统计

直接观察输入法最终提交的中文字符，而不是键盘按键。中文输入量终于接近真实写作产出。

隐私保护

不记录拼音、英文、数字和普通快捷键。跳过密码框、应用黑名单、暂停统计和保留期都能自己设置。

报告与长图

今日、周、月、年报告会把数字翻译成人话，也能导出分享卡片和包含趋势、洞察、词频的长图。

关注词与词库

人名、项目代号、口头禅和自定义词库都能被精确计数，用来观察长期主题和表达习惯。

输入序列

按时间线回看真实上屏片段，按应用和关键词筛选，也能导出 TXT、CSV 或 JSON 做二次分析。

随身鸭

悬浮置顶小窗用指针显示码字速度，配速度曲线、会话字数、今日目标和时钟，写作时不用打开完整看板。

可定制看板与对比

面板可折叠、拖动排序、隐藏；对比报告能并排比较两个时段，使用历史则记录你打开看板和随身鸭的足迹。

词汇量成长 0.3.0

把你累计写过的不同词与不同字，按各自第一次出现的日子累加成一条不断上扬的曲线——直观看见词汇量一路变大。

应用效率与节奏 0.3.0

各应用里你的平均打字速度排行，加上工作日 vs 周末的日均产出对比——看清自己在哪写得最快、节奏更偏平时还是周末。

为什么不一样

数“按键”，和数“真正写下的字”，根本是两回事

中文输入是一串拼音换来一个词——你按了多少键，跟你真正写了多少字几乎无关。普通统计工具数的是前者，DuckType 只数后者。

普通按键统计
ma␣zi␣ya␣2
≈ 12次按键

敲拼音、翻页、选词……每个动作都被算成“输入”，数字虚高，看不出真实产出。

只在输入法把汉字真正“上屏”那一刻才计入，拼音、翻页、英文、数字都不算——这才接近你真实写了多少。

本地优先

统计自己的输入，不变成监控软件

DuckType 的核心数据是每个上屏汉字、时间戳和当时应用名。所有记录默认存在你选择的数据目录里，备份、迁移、删除都由你控制。

  • 不上传输入内容。
  • 不保存拼音、英文、数字。
  • 大段粘贴会被忽略。
  • 常见密码管理器默认黑名单。
  • 可整库导出为 .duckpack 备份。

常见问题

下载前，你可能想先知道这些

它是监控软件吗？会上传我的内容吗？

不会。DuckType 不联网、不上传任何输入内容。它只在本机记录“每个上屏汉字 + 时间 + 当时的应用名”，所有数据都存在你自己选的文件夹里，删除、备份、迁移都由你决定。

它会记录我的密码或具体打了什么吗？

拼音、英文、数字都不入库；大段粘贴会被自动忽略；常见密码管理器默认在黑名单里，密码输入框也会跳过。你可以再自行添加不想统计的程序，或随时暂停。

支持哪些输入法和系统？

目前是 Windows 专属。它对接系统的文本服务框架（TSF）来捕捉上屏汉字，主流拼音 / 五笔等输入法都适用；经典输入框还有一套兜底方案。

杀毒软件 / SmartScreen 报警怎么办？

安装包目前未做数字签名，Windows SmartScreen 可能弹出“Windows 已保护你的电脑 / 未知发布者”。这不代表有病毒，只是还没买代码签名证书、也还没积累下载声誉。放行：在弹窗里点“更多信息 → 仍要运行”。想再确认文件完整，可用每个 Release 附带的 SHA-256 校验值比对（Get-FileHash .\DuckType.exe -Algorithm SHA256），并只从官方 Releases 页下载。介意的话也可以从源码自行构建。

换电脑后数据能带走吗？

可以。在「设置 → 数据管理」里把整库导出成一个 .duckpack 文件，到新电脑导入即可，统计、词库、目标都会一并迁移。

卸载后数据还在吗？要钱吗？

数据在你选择的本地文件夹里，卸载程序不会动它，确认不需要可手动删除。软件本身完全免费、开源（MIT 协议）。
